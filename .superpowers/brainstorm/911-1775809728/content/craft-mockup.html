How It Works

  • Target input — type host/IP at the top, all crafted commands update live
  • Pill buttons (single-select) — scan type, ports, timing — pick one per group
  • Toggle switches (multi-select) — flags like -O, -sV, -A, -v, -Pn, scripts
  • Crafted command — live preview at bottom, color-coded, prominent copy button
  • Custom input — port range shows a text field when "Custom range" is selected
