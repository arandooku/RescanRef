Git

Version control commands

Undo last commit (keep changes) git
git reset --soft HEAD~1 📋 Copy
Moves HEAD back one commit but keeps your changes staged
Interactive rebase last N commits git
git rebase -i HEAD~N 📋 Copy
Replace N with number of commits to rebase
Show file changes in commit git
git show --stat <commit-hash> 📋 Copy

Key Elements

  • Sidebar — app title, search bar, category list, light/dark toggle at bottom
  • Main area — section title + description, then command cards
  • Command card — title, tag badge, code block with one-click copy button, optional description
  • Search — filters across ALL sections, matches title + command text + description
  • Copy button — click to copy command to clipboard, brief "Copied!" feedback
